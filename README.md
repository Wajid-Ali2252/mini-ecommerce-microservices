# Mini E-Commerce Microservices Project

This repository contains a **real-world mini e-commerce microservices system** implemented using **Spring Boot** and **Spring Cloud**.

The project demonstrates:
- Microservices architecture with multiple independent services
- **API Gateway** with centralized authentication (JWT via Spring Security)
- **Config Server** for centralized configuration management
- **Eureka Discovery Server** for service registration and discovery
- **Resilience4j Circuit Breaker** for fault tolerance in inter-service communication
- **User, Product, and Order services** with REST APIs
- **Local Jenkins CI/CD pipeline** to automate build and deployment
- **Postman collection & PPT** for testing and documentation
